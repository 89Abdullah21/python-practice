# Coding Practice Repository

This repository contains a collection of basic to intermediate level coding programs designed to strengthen problem-solving skills and programming fundamentals.

# Overview

The purpose of this repository is to:

Build a strong foundation in python programming
Practice core concepts through hands-on examples
Gradually move from beginner to intermediate level problems

# What’s Included

This repository covers a variety of topics, including:

# 🔹 Basic Concepts
Variables and Data Types
Input/Output Operations
Conditional Statements (if/else)
Loops (for, while)
Basic Functions

# 🔹 Intermediate Concepts
Object-Oriented Programming (OOP)
File Handling 
Error Handling (Exceptions)
Working with APIs (basic level)
Simple Data Structures (Lists, Dictionaries, Sets)

# 🛠️ Technologies Used
Python 🐍 (main language)
Jupyter Notebook

1 Basic python.ipynb contain the hole code. other are mainly Modules

To improve coding skills step-by-step and prepare for real-world development and technical interviews.
